# shadcnui-demo

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app), styled with [Tailwind CSS](https://tailwindcss.com), and using [shadcn/ui](https://ui.shadcn.com) component library.

## Getting Started

First, run the development server:

```bash
npm run dev
# or

# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Features

- [Next.js](https://nextjs.org) App Router
- [Tailwind CSS](https://tailwindcss.com) for utility-first styling
- [shadcn/ui](https://ui.shadcn.com) for accessible, customizable UI components
- Dark mode support
- Example dashboard, authentication, and settings pages
